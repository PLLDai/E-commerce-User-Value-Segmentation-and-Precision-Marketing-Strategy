# E-commerce-User-Value-Segmentation-and-Precision-Marketing-Strategy
1.项目背景
当前电商行业获客成本持续攀升，粗放式运营已无法支撑平台长期增长，精细化用户运营成为提升平台营收、降低用户流失的核心抓手。多数电商平台面临用户画像模糊、价值分层不清晰、运营策略无差别、资源投入ROI低的痛点，无法精准定位高价值用户，也无法针对性解决用户流失问题。
本项目基于电商平台用户全维度行为数据，完成用户全景画像搭建、用户价值分层与行为规律挖掘，为平台精细化运营提供数据支撑与可落地的策略建议。
2.项目核心目标
（1）完成数据清洗与预处理，保障分析数据的准确性与有效性；
（2）进行EDA探索性数据分析，通过数据分布、相关性、行为洞察等搭建平台用户全景画像，明确核心用户群体的人口属性、地域、兴趣与品类偏好特征；
（3）基于RFM经典模型进行优化，构建拥有4个新的优化指标的新模型RFM-I，完成更精准用户价值分层，在8种用户画像的基础上新增7种用户画像；
（4）生成用户画像可视化，进行传统RFM和优化算法的ROI对比测算，直观展示优化算法带来的边际增量，输出分群体、可落地、可验证的精细化运营策略，助力平台提升用户留存、复购与整体营收。
